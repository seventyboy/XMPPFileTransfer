Xmppfiletransfer

java application for file transfer using XMPP

View on GitHub Download .zip Download .tar.gz

Welcome to XMPPFileTransfer: a java GUI Application to upload large files using XMPP bytestream proxy, a service offered by several XMPP networks. Features :

  • built on the top of the smack project ( http://www.igniterealtime.org/projects/smack/);
  • handles interrupted transfers;
  • developed for Windows desktop systems;
  • works behind NAT and firewalls;
  • sender and receiver need to use application at the same time ( as well as peer to peer clients);

application works like peer to peer client, but allows to send file only to your contacts ( friend to friend) . Useful for large file exchanging when computers are behind firewalls
